## 二期 · 想拼工作流 + 图纸识别

**目标**：导入图纸 → 识别材料 → 检查豆仓 → 开拼 → 拼完扣库存。

### 想拼工作流（已实现）

| 功能 | 状态 |
|------|------|
| 导入图纸（上传图片） | ✅ |
| 粘贴材料清单文字识别 | ✅ |
| 手动添加/编辑所需豆子 | ✅ |
| 想拼 / 正在拼 / 拼完了 三区域 | ✅ |
| 选豆仓检查库存是否够拼 | ✅ |
| 不够 → 缺货明细 + 去补库存 | ✅ |
| 够了 → 开始拼（移入正在拼） | ✅ |
| 拼完了 → 自动扣减豆仓库存 | ✅ |
| 完成记录移入拼完了区域 | ✅ |

**流程**：`/wish` 导入 → 想拼区选豆仓检查 → 开始拼 → 正在拼区点拼完了 → 自动扣库存

### 图纸 OCR 识别
